Project Description

A responsive and visually engaging login page built using HTML and CSS. The design features a modern glassmorphism-style form, smooth gradient background animation, and interactive button effects to enhance user experience.

The layout is centered using Flexbox for proper alignment across different screen sizes. Input fields are styled with a minimal underline design and include placeholders for clarity. Subtle hover and transition effects are applied to improve interactivity without overcomplicating the interface.

This project focuses on front-end fundamentals such as layout structuring, CSS animations, and clean UI design practices.

Features:
-Animated gradient background
-Centered login form using Flexbox
-Glassmorphism-style container (blur + transparency)
-Smooth fade-in animation
-Styled input fields with minimal design
-Interactive button hover effects
-Simple and responsive layout

🛠️ Technologies Used
HTML5
CSS3 (Flexbox, animations, transitions)

⚠️ Limitations

This is a front-end only project. No backend functionality, authentication system, or form validation is implemented.
